# MLP from scratch

Implémentation d'un perceptron multi-couches from scratch avec juste NumPy,
testé sur Fashion-MNIST (70 000 images de vêtements, 10 classes).

87% d'accuracy sur le test set.

## Lancer le script
```
pip install numpy matplotlib scikit-learn
python mlp_from_scratch.py
```

## Structure
- `mlp_from_scratch.py` : le script
- `mlp_from_scratch.ipynb` : le notebook avec les explications
